# AImigo

### Planning Background
In the case of students with overlapping COVID times, we found it difficult to complete learning in the first and second grades of building a lot of concepts, and we found the problem that using it too much like a habit often forgets its conceptual meaning.</br>
So, we decided to plan the development of the application because I hope that learning would be fun.
### Meaning of app name
It is a combination of AI and amigo, which means friend in Spanish.</br>
We hope this app will be your friend by taking a quiz related to AI.

### App Screen Configuration
You can switch between screens by swiping.


#### Main (Home)
- You can see the character you customized in the Closet screen.
- If the user is inactive for more than 2 days, the character’s facial expression changes.
- If you exit while solving a quiz, you can continue solving it using the button below.
- You can add friends through the **Add Friends** button in the top-right corner.
- You can change or check settings such as nickname, sound, and help through the **Settings** button.


#### Left (Quiz)
- You can solve quizzes on various AI and coding-related subjects.
- You can choose the difficulty level according to your skill level.
- Higher-level subjects are unlocked only after clearing prerequisite subtopics.
- Quiz difficulty is automatically adjusted using an internal algorithm based on the user’s achievement level.


#### Right (Closet)
- You can customize your character with clothes and hats.
- The customized appearance is reflected on the Home screen and during quizzes.
- You can purchase clothes using coins earned by solving quizzes.
